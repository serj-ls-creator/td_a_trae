# Текущий статус проекта: Neon Pixel Tower Defense (Kuromi Style)

## 1. Технологический стек
- **Движок**: Phaser 3.90.0 (WebGL)
- **Сборка**: Vite + TypeScript
- **Стилистика**: Neon Pixel Art / Kuromi Style (фиолетовый неон, черный фон, эмодзи-спрайты)

## 2. Реализованные механики

### **Карта и Мир**
- **Изометрическая сетка**: 10x10 плиток с автоматической сортировкой глубины (`setDepth(y)`).
- **Z-образный путь**: Фиксированный маршрут для монстров, выделенный розовым неоновым свечением.
- **Процедурный декор**: Сверхплотная застройка (95% свободных клеток) с несколькими объектами (домиками и деревьями) на одну клетку.
- **Баланс башен**: Радиус поражения уменьшен в 3 раза для всех типов башен.
- **Сетка занятости**: Система `isOccupied`, блокирующая установку башен на здания или дорогу. Декор теперь может спавниться вплотную к дороге.

### **Боевая система**
- **4 типа башен**: 🌸 (Flower), 🐒 (Monkey), 👻 (Ghost), 🎐 (Fan) с уникальными характеристиками.
- **Уникальные снаряды**: ✨, 🍌, 🔮, 💨 для каждой башни соответственно.
- **ИИ врагов**: Поиск ближайшей цели в радиусе, кулдаун стрельбы.
- **Визуальные эффекты**: 
    - Неоновое свечение (`PostFX Glow`) для всех башен и снарядов.
    - Flash-эффект (белый цвет) при получении урона монстрами.
    - Анимация смерти (вращение и уменьшение).

### **Экономика и Прогресс**
- **Система волн**: 10 волн с постепенным усложнением.
- **Типы врагов**: 😈 (Slime), 💀 (Skeleton), 🧛 (Bat) с разным HP и скоростью.
- **Ресурсы**: Золото (за убийства) и Жизни (Lives).

### **Интерфейс (UI)**
- **Главный герой**: Программная отрисовка Куроми в конце пути.
- **Обратная связь**: Тряска персонажа и камеры при пропуске врага.
- **Магазин башен**: Полупрозрачная панель с неоновой рамкой и "призраком" башни при выборе.
- **Интерактивность**: 
    - Плитки подсвечиваются при наведении (белым для дороги, фиолетовым/красным для земли).
    - Установленные башни показывают радиус поражения при наведении мыши.
    - "Призрак" башни при выборе из меню теперь также показывает радиус поражения для удобства планирования.
